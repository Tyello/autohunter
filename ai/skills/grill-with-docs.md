# Skill: grill-with-docs

Use para confrontar um plano contra as docs vivas antes de implementar.

## Objetivo

Evitar que o agente implemente algo que contradiz decisões já tomadas.

## Docs obrigatórias

- `AGENTS.md`
- `README.md`
- `docs/LLM_CONTEXT.md`
- `docs/ARCHITECTURE.md`
- `docs/PROJECT_GUIDELINE.md`
- `docs/ROADMAP.md`
- `docs/LAUNCH_PLAN.md` quando envolver lançamento
- `docs/AUCTION_RUNTIME.md` quando envolver leilões

## Processo

1. Leia o plano.
2. Leia as docs relevantes.
3. Liste inconsistências.
4. Liste riscos e decisões já tomadas.
5. Ajuste o plano antes de implementar.
6. Indique se o plano deve seguir, mudar ou ser descartado.

## Saída esperada

- Decisões confirmadas.
- Contradições encontradas.
- Ajustes obrigatórios.
- Riscos remanescentes.
