# Skill: handoff

Use para compactar uma conversa longa em contexto reutilizável para outro chat/agente.

## Objetivo

Gerar contexto confiável e copiável sem depender de memória informal.

## Estrutura

Inclua:

- Estado atual confirmado.
- Decisões tomadas.
- Arquivos alterados ou relevantes.
- Problemas resolvidos.
- Pendências.
- Riscos.
- Próximos passos recomendados.
- Comandos de validação.
- Pontos que não devem ser reabertos sem evidência nova.

## Regras AutoHunter

- Diferencie fato confirmado de hipótese.
- Não declare como implementado algo apenas planejado.
- Preserve decisões de produto já tomadas.
- Cite docs relevantes quando possível.

## Saída esperada

Um bloco único, copiável para novo chat ou issue.
